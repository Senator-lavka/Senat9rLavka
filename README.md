# Лавка Сенатора

Обновление: галактический дизайн, слоган «Самые дешевые цены в галактике!», длинные карточки для полного фото товара, редактирование товаров, добавление и удаление фото, бронь товара и снятие брони в админке.

## Важно для Supabase
Если после заказа ошибка про `reserved_qty`, выполни `supabase.sql` в Supabase SQL Editor.

Для удаления фото из товара дополнительный SQL не нужен: фото удаляется из списка товара в базе. Сам файл в Storage может оставаться, это нормально.
