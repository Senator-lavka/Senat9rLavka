# Лавка Сенатора

Telegram Mini App магазин на React + Vite + Supabase.

## Что добавлено в этой версии

- Подтверждение заказа больше не снимает бронь.
- Кнопка **«Выдан / оплачен»** списывает заказанное количество из остатка и снимает бронь.
- Кнопка **«Отменить»** снимает бронь без списания остатка.
- Для товаров с остатком `0` появляется кнопка **«＋ Жду этот товар»**.
- В админке появилась вкладка **«Ждут товар»** и отметка количества ожиданий у каждого товара.
- Фото товара можно листать свайпом по экрану.

## Важно после загрузки

1. Загрузить содержимое архива в GitHub поверх старых файлов.
2. Удалить `package-lock.json`, если он снова появился.
3. В Supabase выполнить `supabase.sql` через SQL Editor.
4. В Vercel сделать Redeploy без Build Cache.
