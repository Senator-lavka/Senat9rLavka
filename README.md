# Лавка Сенатора

Обновление: редактирование товаров, добавление фото к существующим товарам, бронь при заказе и кнопка «Убрать бронь» в админке.

## Важно после загрузки этой версии

В Supabase → SQL Editor выполни `supabase.sql` из этого архива. Это добавит поле `reserved_qty` и политики доступа.

## Как работает бронь

- Если товара 1 штука и покупатель оформил заказ на 1 — всем остальным будет видно «Забронировано».
- Если товара 2 штуки и покупатель оформил заказ на 1 — будет доступна ещё 1 штука, бронь не перекроет товар.
- В админке у товара есть кнопка «Убрать бронь».
