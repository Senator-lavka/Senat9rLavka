# Лавка Сенатора

Telegram Mini App магазин на React + Vite + Supabase.

## Важно для Vercel

В архиве специально нет `package-lock.json`, чтобы Vercel не тянул зависимости из неправильного lock-файла. Загружай в GitHub содержимое архива целиком.

## После загрузки файлов

1. GitHub: загрузи содержимое архива поверх старых файлов.
2. Vercel: дождись деплоя. Если сборка падает из-за npm-сети, сделай Redeploy without Build Cache.
3. Supabase: открой `supabase.sql`, скопируй весь текст и выполни в SQL Editor.

## Что добавлено

- Раздел «Заказы» в админке.
- Бронь товара на 30 минут после оформления заказа.
- Статусы заказов: ожидает, подтверждён, выдан/оплачен, отменён, истёк.
- Автоматическое снятие просроченной брони при открытии/обновлении приложения.
- Счётчик «Всего заработано» — считает только заказы со статусом «Выдан / оплачен».
- Кнопка «Предложить товар» в шапке.
