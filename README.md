# Лавка Сенатора

React/Vite Telegram Mini App магазин.

## Что добавлено
- Галактический стиль.
- Корзина и бронь товара.
- Админка с редактированием товара.
- Добавление и удаление отдельных фото товара.
- Кнопка **«Предложить товар»** для покупателей.
- В админке вкладка **«Предложения»** со счётчиком новых предложений.
- Текст для покупателя: **«Ваше предложение не гарантирует наличие товара в будущем.»**

## После загрузки в GitHub
1. Дождись деплоя Vercel.
2. Открой Supabase → SQL Editor.
3. Выполни `supabase.sql`, чтобы создалась таблица `product_suggestions` и политики доступа.
